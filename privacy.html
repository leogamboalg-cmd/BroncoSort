🔒 Privacy & Security

Your data
stays yours.

BroncoSort is built to improve class registration — not collect unnecessary data. The extension only accesses information required to display professor ratings, sort sections, and process school compatibility requests.

BroncoSort Privacy Minimal Data Collection

Collected

Professor names, visible course section information, and optional school support requests.

Never Collected

Passwords, payment information, transcripts, emails, or browsing activity outside supported school pages.

Storage

BroncoSort minimizes stored information and only uses Chrome extension storage for settings and supported functionality.

What BroncoSort accesses

BroncoSort only accesses information necessary to provide professor ratings, sorting functionality, and school compatibility requests.

  • Professor names visible on registration pages
  • Course section information already displayed in your browser
  • School request submissions for unsupported campuses

What BroncoSort does not access

  • Passwords or login credentials
  • Payment information
  • Personal messages or emails
  • Browsing history outside supported school pages
  • Student records or transcripts

How school requests work

When students request support for a new campus, BroncoSort may temporarily receive limited registration page structure information to improve compatibility for that school.

Third-party services

BroncoSort uses publicly available professor rating information from RateMyProfessors. BroncoSort is not affiliated with RateMyProfessors.

Local storage

Settings and preferences may be stored locally using Chrome extension storage to preserve extension functionality.

Questions or concerns?

If you have questions regarding privacy, data handling, or extension permissions, please contact the BroncoSort developer through the Chrome Web Store listing.
